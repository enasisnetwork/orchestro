Bases: Config

Contain the configurations from the arguments and files.

Parameters:

  • sargs ‚Äì Additional arguments on the command line.

  • files ‚Äì Complete or relative path to config files.

  • paths ‚Äì Complete or relative path to config paths.

  • cargs ‚Äì Configuration arguments in dictionary form, which will override contents from the config files.
